## 🏨 Hotel UQ

Software para la gestión de reservas de habitaciones del hotel “Descanso Feliz”. El hotel se compone de 20 habitaciones, cada Habitación tiene un número, capacidad, disponibilidad y un tipo (sencilla, doble, lujo). En la GUI debe aparecer una matriz (de tamaño 4x5) de botones donde el administrador del hotel registra una reserva de habitación. Al presionar el botón debe aparecer una nueva ventana donde se registran los datos de la reserva: código (que se crea aleatoriamente), la fecha de ingreso, la fecha de salida, cantidad de personas y la información del usuario, un usuario tiene una cédula, nombre, correo.

### Tecnologías

- Java 21
- JavaFX 21
- Maven
